## 1. Глоссарий

| Термин | Определение |
|--------|-------------|
| Перерыв (break) | Период, когда пользователю предлагается отвлечься от экрана |
| Короткий перерыв | Перерыв длительностью 20 секунд по правилу 20-20-20 |
| Длинный перерыв | Перерыв длительностью 5 минут после продолжительной работы |
| Рабочий интервал | Время непрерывной работы между перерывами |
| Overlay | Полупрозрачное окно поверх всех окон с таймером перерыва |
| Idle | Состояние, когда пользователь не взаимодействует с компьютером |
| Idle threshold | Порог времени бездействия, после которого таймер сбрасывается |
| Snooze | Откладывание перерыва на заданное время |
| Skip | Полный пропуск текущего перерыва |
| FSM | Finite State Machine — конечный автомат |
| Tray | Область системных иконок в панели задач |
| DE | Desktop Environment — окружение рабочего стола (GNOME, KDE) |

## 2. Целевая аудитория

### 2.1 Основные пользователи

- Разработчики, проводящие 6+ часов за экраном ежедневно
- Дизайнеры и контент-мейкеры с длительными рабочими сессиями
- Студенты, работающие за компьютером с учебными материалами
- Любые пользователи Linux, заботящиеся о здоровье глаз

### 2.2 Профиль пользователя

- Операционная система: Linux (Ubuntu, Fedora, Arch, Debian)
- Окружение рабочего стола: GNOME, KDE Plasma
- Протокол отображения: Wayland (основной), X11 (совместимость)
- Техническая подготовка: средняя и выше (умеет запустить AppImage)
- Языки интерфейса: английский (v1.0), русский (v1.1)

## 6. Ограничения и допущения

### 6.1 Ограничения

| ID | Ограничение |
|----|------------|
| CON-01 | Приложение работает только на Linux. macOS и Windows — в roadmap v2.0 |
| CON-02 | Интерфейс v1.0 только на английском языке |
| CON-03 | Wayland: ограниченная поддержка global hotkeys из-за ограничений протокола |
| CON-04 | Overlay может не отображаться поверх некоторых fullscreen-приложений на Wayland |
| CON-05 | Нет сетевой синхронизации данных между устройствами |
| CON-06 | GNOME не поддерживает нативный tray без расширения AppIndicator |
| CON-07 | Flatpak sandbox ограничивает доступ к idle detection — требуются дополнительные порталы |

### 6.2 Допущения

| ID | Допущение |
|----|----------|
| ASM-01 | У пользователя установлен один из поддерживаемых DE (GNOME, KDE) |
| ASM-02 | D-Bus session bus доступен (стандарт для desktop Linux) |
| ASM-03 | Пользователь имеет права на запись в `$XDG_DATA_HOME` |
| ASM-04 | На Wayland доступен протокол `ext_idle_notify_v1` (поддерживается GNOME 45+, KDE 6+) |
| ASM-05 | Пользователь может самостоятельно запустить AppImage (chmod +x, double-click) |
| ASM-06 | На системе установлен libwebkit2gtk (Tauri runtime dependency) |

## 7. Приоритизация (MoSCoW)

### Must have — без этого приложение не имеет смысла

- Рабочий таймер с настраиваемыми интервалами (FR-100–FR-106, FR-108–FR-109)
- Idle detection с автосбросом (FR-200–FR-202, FR-206)
- Overlay с таймером и кнопками (FR-300–FR-303, FR-306, FR-308)
- System tray с меню (FR-400, FR-402–FR-403, FR-405–FR-406)
- Настройки с валидацией и персистентностью (FR-500–FR-503, FR-506, FR-509–FR-512)
- Базовая статистика (FR-600–FR-603, FR-605–FR-606)
- Производительность < 50 MB RAM, 0% CPU idle (NFR-100–NFR-101)
- Работа на GNOME Wayland + X11 fallback (NFR-300, NFR-302)
- AppImage как формат распространения (NFR-304)

### Should have — существенно улучшает продукт

- Длинные перерывы с отдельным интервалом (FR-101, FR-103, FR-110)
- Плавные анимации overlay (FR-304–FR-305)
- Иконки состояния в tray (FR-401, FR-404)
- Snooze и idle threshold в настройках (FR-504–FR-505)
- Звук перерыва (FR-800)
- Автозапуск (FR-900–FR-902)
- Тёмная тема (FR-508)
- Нотификации как fallback (FR-700, FR-703)
- KDE Plasma поддержка (NFR-301)
- `.deb` пакет (NFR-305)

### Could have — приятное дополнение

- Мотивационные сообщения в overlay (FR-307)
- Кнопки действий в нотификации (FR-701)
- Bar chart в статистике (FR-604)
- Fullscreen detection (FR-310)
- Flatpak (NFR-306)

### Won't have (v1.0) — отложено

- Мультиязычный интерфейс
- Упражнения для глаз в overlay
- Pomodoro-режим
- Экспорт статистики
- macOS / Windows сборка
- Синхронизация между устройствами
