# LookAway for Linux — План разработки

> Пошаговый план от нуля до релиза. Каждая фаза — автономный инкремент с работающим результатом.

---

## Общие принципы планирования

- Каждая фаза заканчивается работающей сборкой. Никаких «подготовительных» фаз без результата.
- Фаза не начинается пока предыдущая не пройдена и не протестирована.
- Оценки — в рабочих днях одного разработчика. Буфер 20% заложен в каждую фазу.
- Приоритет: сначала функциональное ядро, потом UX-полировка, потом расширения.

## Пост-релиз roadmap

Фичи, которые не входят в v1.0, но запланированы:

### v1.1 — Расширения перерывов

- Упражнения для глаз (анимированные инструкции в overlay)
- Разминка: подсказки по растяжке шеи, спины, рук
- Дыхательные упражнения с визуальным ритмом

### v1.2 — Продвинутая статистика

- Экспорт в CSV
- Графики за месяц / год
- Streak (серия дней без пропусков)
- Сравнение недель

### v1.3 — Интеграции

- Flatpak упаковка с sandbox-разрешениями
- D-Bus API для интеграции с другими приложениями
- Pomodoro-режим как альтернативная стратегия таймеров

### v2.0 — Мультиплатформенность

- macOS сборка (замена Look Away)
- Windows сборка
- Синхронизация настроек между устройствами (опционально, opt-in)
